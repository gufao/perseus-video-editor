# FFmpeg Binaries

Please download the static binaries for FFmpeg and place the executable in the corresponding folder.

- **macOS**: [Download from evermeet.cx](https://evermeet.cx/ffmpeg/) -> put `ffmpeg` in `ffmpeg-mac/`
- **Windows**: [Download from gyan.dev](https://www.gyan.dev/ffmpeg/builds/) -> put `ffmpeg.exe` in `ffmpeg-win/`
- **Linux**: [Download from johnvansickle.com](https://johnvansickle.com/ffmpeg/) -> put `ffmpeg` in `ffmpeg-linux/`

Ensure the binaries are executable (`chmod +x ffmpeg` on Mac/Linux).
